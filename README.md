# Описание проекта
Этот  прект  демонстрирует все этапы решения классической DS задачи регресии. В качестве примера взят набор данных, который содержит цены и другие характеристики почти 54 000 бриллиантов., датасет взят отсюда: https://www.kaggle.com/datasets/shivam2503/diamonds?resource=download
Проект включает в себя следующие этапы:

Проверка на корректность и качество данных (файл baseline.ipynb)
Детальное изучение признаков и таргета (файл baseline.ipynb)
Изучение взаимосвязей между признаками (файл baseline.ipynb)
Препроцессинг данных (файл baseline.ipynb):
Обработка пропущенных значений
Кодирование категориальных переменных
Приведение данных к единому типу
Построение базовой модели и подсчет метрик (файл baseline.ipynb)
Преобразования данных, тестирование с помощью кросс-валидации. Выбор лучших преобразований для улучшения предсказаний (файл improvements.ipynb)
Подбор гиперпараметров разными методами (файл hyperparam_tuning.ipynb)

eda.ipynb
В этом файле были удалены некорректные наблюдения, разделили на категориальные и количественные признаки. Явных выбросов на графиках не было обнаружено. На круговой диаграмме: Большинство бриллиантов считаются идеальными 39.9% Далее идут премиальные 25.6% и очень хорошие 22.4%

 "Цвет бриллианта, где D — лучший, а J — худший." D 12.7%, a J 5.21%, но больше всего бриллиантов в цвете G 20.9%

 "Чистота — показатель степени прозрачности алмаза (I1 (худший), SI2, SI1, VS2, VS1, VVS2, VVS1, IF (лучший))" Большинство SI1 24.2%, VS2 22.7%. Самый лучший IF всего 3.32%
 Так же "Цена очень сильно коррелирует с каратом (весом алмаза), x, y, z (Длины граней алмаза). Самая низкая корреляция с глубиной и цветом."

 baseline.ipynb
 В этом файле разделили на тренировочную и тестовую выборки. Были использованы OrdinalEncoder и OneHotEncoder. Были использованы модели: DummyRegressor, LinearRegression, DecisionTreeRegressor, RandomForestRegressor, CatBoostRegressor. Catboost оказался лучшей моделью с mae_mean 270.187135 и r2 0.981826

improvements.ipynb 
В этом файле были проведены эксперименты, в ходе которых было выяснено, что только первый эксперимент не показал метрику выше

hyperparam_tuning.ipynb
В этом файле производился подбор гиперпараметров для лучшей модели: depth=10, iterations=700, l2_leaf_reg=1, learning_rate=0.05, loss_function='RMSE', random_state=12345, random_strength=1, verbose=0
