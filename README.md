# Описание проекта
Этот  прект  демонстрирует все этапы решения классической DS задачи регресии. В качестве примера взят набор данных, который содержит цены и другие характеристики почти 54 000 бриллиантов., датасет взят отсюда: https://www.kaggle.com/datasets/shivam2503/diamonds?resource=download
Проект включает в себя следующие этапы:

Проверка на корректность и качество данных (файл baseline.ipynb)
Детальное изучение признаков и таргета (файл baseline.ipynb)
Изучение взаимосвязей между признаками (файл baseline.ipynb)
Препроцессинг данных (файл baseline.ipynb):
Обработка пропущенных значений
Кодирование категориальных переменных
Приведение данных к единому типу
Построение базовой модели и подсчет метрик (файл baseline.ipynb)
Преобразования данных, тестирование с помощью кросс-валидации. Выбор лучших преобразований для улучшения предсказаний (файл improvements_cv.ipynb)
Подбор гиперпараметров разными методами (файл hyperparam_tuning.ipynb)
